# Simple Telegram Bot
An example of Telegram bot using  [Python](https://www.python.org/ "Python").

# Installation
With package manager [pip](https://pip.pypa.io/en/stable/ "pip") install libraries  in `requirements.txt` . <br />
Edit `config.json`  with your data.
